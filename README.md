### Installation

1. Clone this repository:
```bash
git clone <your-repo-url>
cd chatgpt-js
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory and add your OpenAI API key:
```
OPENAI_API_KEY=your_api_key_here
```

### Usage

Run the application:
```bash
node index.js
```
